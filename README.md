A) Text-to-Speech Converter Using Google Text-to-Speech (gTTS) (main.py)

1) Overview

This project allows you to convert text from various input formats (text, PDF, Word, ODT) into speech using Google Text-to-Speech (gTTS). Users can select their input format, either input the text directly or upload a file, and then play the converted speech through a media player. The program provides a simple and user-friendly GUI, making it accessible for non-technical users.

2) Features

Supports input in the form of plain text, PDF, Word (docx), or ODT files.
Uses the Google Text-to-Speech (gTTS) engine for speech synthesis.
Plays the synthesized speech automatically using pygame.
Easy-to-use graphical interface for selecting input format and entering text.

3) Dependencies

The following Python libraries are required:
- gtts
- pygame
- PyPDF2
- python-docx
- odfpy
- tkinter

4) Usage

- Run the script.
- Select the input format (Text, PDF, Word, or ODT).
- Either enter the text directly or upload the selected file type.
- The program will synthesize speech and play it using the selected gTTS voice.
